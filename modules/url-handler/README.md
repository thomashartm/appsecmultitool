# URL Handler

Parses and splits up a URL into a consumable and 
processable object which allows to access the single building blocks of the URL.

The generated URL object provides the following properties: 
* source    
* protocol    
* authority    
* userInfo    
* user    
* password    
* host    
* port    
* relative    
* path    
* directory    
* file    
* query    
* anchor
